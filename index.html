Bienvenue sur la page de SCWINDEB

Nous vous présentons un outil de débogage pour MS Windows que vous pouvez utiliser dans vos mission de Criminalistique et de REM à la place de gdb ou PyDbg.

Comment l'utiliser

identifiez le PID(Process Identifier) dans le gestionnaire de tâches windows. Sur un terminal ou un prompt shel cmd.exe, exécutez le script comme suit:

C:>SCWINDEB\ python SCWINDEB_run.py pid

Quand le message "Press a key to continue..." est affiché sur votre prompt, vous ne pouvez plus interagir avec votre application que vous essayez de déboguer. Ceci est dû au fait que le processus corespondant est suspendu.
